# Options Pricing Engine

A quantitative finance project for European option pricing, Greeks,
implied volatility estimation, and volatility surface analysis.

## Current Progress

- Project structure created
- Python virtual environment configured
- NumPy-based call and put intrinsic value utilities implemented

## Planned Features

- Black-Scholes pricing
- Greeks
- Implied volatility solver
- Option-chain data cleaning
- Volatility smile and surface visualization
- Unit tests
